vue基础语法
  • {{}}的作用
    • 打印字面量或者变量，可以写运算符用于计算数值或拼接字符串
  • v-html
    • 将字面量或变量当html字符拼接在内部
  • v-text
    • 打印字面量或者变量
  • v-pre
    • 打印符号本身
  • v-once
    • 显示值,不可通过更改数据进行更改
  • v-cloak
    • 空白代替延迟期间的内容
  • v-bind
    • 绑定属性
  • computed
    • 调用方法处理数据赋值给某个属性
  • v-filter
    • 过滤器，数据|方法 的方法
  • data响应的方法
    • splice
      • splice(位置，删除数量，替换字符)
    • unshift
      • 最前面添加字符
    • push
      • 最后面添加数据
    • pop
      • 删除最后一个数据
    • shift
      • 删除第一个数据
    • sort
      • 顺序排列
    • reverse
      • 倒序排列
  • v-for（能遍历什么？）
    • 遍历数组
    • 遍历对象
      • (val,key,index) in items
  • v-for（特性是什么？）
    • 将index或item绑定给属性key可以帮助重用增加效率
  • v-on（功能是什么）
    • 触发事件调用方法
  • v-on（特性是什么）
    • 没有括号默认获取event参数
    • 可以获取event参数和其他参数（方法）
      • (参数,$event)
    • 通过修饰符限制触发默认或者自定义的事件（修饰符的特性）
      • 所有的行为都可以串联
  • v-on有哪些修饰符
    • .stop
      • 调用 event.stopPropagation()。
      • 分离内外圈
    • .prevent
      • 调用 event.preventDefault()。
      • 阻止默认事件，只允许函数内的事件
    • .{keyCode | keyAlias}
      • 只当事件是从特定键触发时才触发回调。
      • 对按键做出反应
    • .native
      • 监听组件根元素的原生事件。
      • 组件中用到，组件名作的模板点击触发使用
    • .once
      • 只触发一次回调。
  • v-model
    • 双向绑定数据
    • input type="text"
      • 绑定data的某个文本属性，通过输入改变这个文本属性
    • input type="radio"
      • 绑定data的某个布尔属性，如sex,通过true和false决定改变按钮状态，通过点击按钮调整这个布尔值
    • <input type="checkbox"<"radio">><select>
      • 绑定data的某个布尔属性控制单个复选框是否被选定，这个复选框调整这个布尔值
      • 绑定data的某个列表，控制都个复选单元是否被选中，复选项调整列表内容
    • select
      • 多选需要加什么
        • multiple
      • value放在哪里
        • option

    • lazy修饰符：
      • 可以让数据在失去焦点或者回车时才会更新
    • number修饰符：
      • 可以让在输入框中输入的内容自动转成数字类型：
    • trim修饰符：
      • 可以过滤内容左右两边的空格
